Description

The buzz around Angular and the relevance front end framework in the market, this is a critical focus area. This course is mainly for professionals who are currently working as web developer and want to enhance their skills with Angular.


Prerequisites

This course starts from the scratch and you do not require any prior knowledge of Angular.

Setup & installation

 Node.js
 Angular cli
 Visual Studio Code
